# GUI Unit Converter (Python)

## About

A simple GUI Unit Converter built using Python and Tkinter.
This application converts common units such as distance, temperature, and weight.

---

## ✨ Features

✔️ Kilometers to Miles conversion
✔️ Celsius to Fahrenheit conversion
✔️ Kilograms to Pounds conversion
✔️ Input validation
✔️ Simple and user-friendly GUI

---

## 🛠️ Tech Used

* Python
* Tkinter

---

## 🚀 How to Run

1. Download the project
2. Open the Python file
3. Run the program
4. Enter a value
5. Select conversion type
6. Click **Convert**
